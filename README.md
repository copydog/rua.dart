# Rua Framework
The Flutter Framework For App Artisans.

### Features
1. Annotation-based.
1. Compile-time IoC.
1. Compile-time Aop.
1. All-in-one out-of-box production-ready framework.

### Inspiration
Most ideas comes from [Spring Framework](https://spring.io/)

### Included Packages
1. IoC
1. Aop
1. Config
1. Rest
1. View
1. Route
1. Handler
1. Repository (MobX)
1. Rson (Json De/Serializer)
1. Initializer
1. Stream
1. State Machine
1. EventBus
